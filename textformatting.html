Text Formatting Tags in HTML

Definition:

These tags help you emphasize, define, or style text semantically, meaning they give meaning to the text, which is also helpful for accessibility and SEO

Different types of Text Formatting tags

Strong Tag

The Strong tag indicates strong importance or seriousness. Usually rendered bold by browsers

Emphasize Tag:

It indicates emphasis (stress) on the text. It usually rendered as italic. The screen readers may change voice tone.

Mark Tag:

It Highlights or marks text (like using a highlighter pen). It usually rendered with a yellow background

Small Tag

It represents side comments or fine print. It usually rendered as smaller text

Definition Tag

It represents the defining instance of a term (the term being defined). It is typically shown in italics. it is useful in glossaries or definition.

Definition of Block level elements

Block Level Element start on a new line and take up the full width available (by default) It can contain other block-level elements and inline elements. It create rectangular blocks in the layout

Quotation Tag:

There are 2 methods of writing Quotations
1. Q tag: q tag is used for short, inline quotations inside a paragraph. the vbrowsers usually add quotation marks automatically.
2. Blockquote tag: It is used for longer quotations displayed as a separate block (usually indented). It can include multiple paragraphs. It often used to quote external sources or large chunks of text.
